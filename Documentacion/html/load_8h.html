◆ cargarJuego()

void cargarJuego ( )

Carga los datos de configuración del juego desde el archivo Juego.txt.

Carga un juego desde archivo.

Esta función realiza las siguientes acciones:

  • Carga el tamaño del tablero, número de barcos y tipos de barcos.
  • Carga las cantidades de cada tipo de barco.
  • Carga los datos de los jugadores (ID, nombre, tipo de disparo, etc.).
  • Carga las matrices de los tableros (flota y oponente) de cada jugador.
Note
El archivo debe estar correctamente formateado para evitar errores de lectura.
Warning
El programa finaliza si no puede abrir el archivo correctamente.
